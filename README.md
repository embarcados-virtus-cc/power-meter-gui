# CONFIGURAÇÕES DA APLICAÇÃO:

## Comandos importantes para desenvolvimento da aplicação:

- Instalar as dependências (pnpm é o gerenciador de pacotes padrão que estamos usando):
```bash
pnpm install
```
- Rodar o servidor (Vite):
```bash
pnpm dev 
```
- Testar a aplicação (Vitest):
```bash
pnpm test
```
- Rodar o linter (ESLint):
```bash
pnpm lint
```
- Formatar através do linter (ESLint):
```bash
pnpm format
```
- Verificar sanidade do código (ESLint):
```bash
pnpm run check
```

Outras configurações relacionadas a desenvolver código de fato, estão no arquivo de [SETUP](./SETUP.md)
